# Global Regions Feature

## Overview
The dashboard now displays all 28 AWS regions worldwide with their carbon intensity and renewable energy percentages.

## Features

### 1. Worldwide View
- **28 AWS Regions**: Complete coverage across all continents
- **Clean Grid Layout**: Responsive design that adapts to screen size
- **Non-cluttered**: Compact tiles showing essential information

### 2. Information Displayed on Each Tile
- **Region Code**: AWS region identifier (e.g., `eu-north-1`)
- **Location**: City and country (e.g., "Stockholm, Sweden")
- **Carbon Intensity**: Data center intensity in gCO₂/kWh (accounting for AWS renewable energy)
- **Renewable Energy**: AWS renewable energy percentage with ⚡ icon
- **Color Coding**: Visual indication of carbon intensity level
  - Green: Very Low (≤50)
  - Light Green: Low (≤150)
  - Grey: Moderate (≤250)
  - Orange: High (≤400)
  - Red: Very High (>400)

### 3. Interactive Features
- **Click to View Details**: Click any region tile to see detailed information
- **Modal with Full Data**:
  - Grid intensity vs. data center intensity
  - AWS renewable energy impact
  - Comparison with optimal region
  - Potential savings percentage

## Data Sources

### Primary: API Endpoint
- **Endpoint**: `GET /global-regions`
- **Real-time**: Fetches live data from AWS Global Carbon Optimizer
- **Includes**: UK real-time data when available

### Fallback: Static JSON
- **File**: `public/global-regions.json`
- **Generated by**: `generate_global_regions_json.py`
- **Update**: Run script to refresh data

## Generating Static Data

```bash
# From project root
python green-qa-platform/lambda/carbon_ingestion/generate_global_regions_json.py
```

This will:
1. Fetch data from all 28 AWS regions
2. Calculate data center intensities with AWS renewable energy
3. Save to `dashboard/public/global-regions.json`
4. Display summary statistics

## Navigation

The dashboard now has 6 sections:
1. **Dashboard**: Overview and statistics
2. **Europe**: European AWS regions (8 regions)
3. **Worldwide**: All AWS regions globally (28 regions)
4. **Forecast**: UK 48-hour forecast (when available)
5. **Calculator**: Carbon footprint calculator
6. **History**: Test execution history

Running workloads in **eu-north-1** instead of **af-south-1** can save:
- **510.1 gCO₂/kWh** (99.9% reduction)
- For a 1-hour test with 2 vCPUs: ~500g CO₂ saved

## Technical Implementation

### Frontend (app.js)
- `loadGlobalRegions()`: Fetches data from API or static file
- `renderGlobalRegionGrid()`: Renders tiles with color coding
- `openGlobalRegionModal()`: Shows detailed region information

### Backend (handler.py)
- `/global-regions` endpoint: Returns all regions with carbon data
- Integrates with `aws_global_carbon_optimizer.py`

### Styling (styles.css)
- `.global-region-grid`: Responsive grid layout
- `.global-region-item`: Compact tile design
- `.renewable-icon`: Visual indicator for renewable energy

## Browser Compatibility
- Modern browsers (Chrome, Firefox, Safari, Edge)

- Grid layout with automatic column sizing

- Efficient rendering: Only updates when data changes
